## 要闻

### 1. 阿里千问上线 2.4T 参数旗舰模型预览版 Qwen3.8-Max-Preview

阿里正式上线新一代旗舰模型预览版 **Qwen3.8-Max-Preview**，参数量达 **2.4T**。该模型已在 Qwen Studio、Token Plan、Qoder 系列以及千问 PC 端等平台上线，正式版预计近期发布并同步上线千问 APP。

官方称该模型较前代 Qwen3.7-Max 有明显提升，正在以“天”为单位持续进化，在代码工程和专业办公等核心领域展现出接近世界前沿模型的实力。Qoder 调用该模型目前有白天 1 折、夜间最低 0.2 折的限时优惠。另有博主公布疑似内部基准测试，称其编码能力仅落后于 Fable 5，智能体任务表现优于 Opus 4.8 Max。

### 2. 千问 AI 平台发布 Token Plan 个人版

千问 AI 平台发布 **Token Plan 个人版**订阅服务，同步上线 Qwen3.8-Max-Preview。个人版提供 Lite、Standard、Pro 三档，采用 **5 小时**与 **7 天**两层滑动窗口 Credits 限额，面向 Cursor、Claude Code 等交互式编程与 Agent 工具场景。

官方称 Qwen3.8-Max-Preview 在该服务中目前享白天 1 折、夜间 0.2 折的限时优惠。

### 3. Kimi 因 K3 需求激增暂停新用户订阅

月之暗面 Kimi 团队称，Kimi K3 发布后 48 小时内用户请求量大幅超出预估，接近现有集群承载极限。团队已暂停 C 端新用户订阅，将算力优先用于保障已订阅用户权益。

官方表示正在全速扩容，后续将分批恢复新订阅名额。未来新订阅用户的会员权益计划拆分为 **Kimi 主权益**（Kimi Web、Kimi APP、Kimi Work）和 **Kimi Code 权益**两套方案，以便更精确匹配算力资源。

- https://www.kimi.com/membership/pricing?from=upgrade_plan

### 4. Kimi 旧套餐用户仍可续订旧套餐

Kimi 曾短暂上线展示拆分后的新会员体系，但随后为方便未到期及关闭自动续费的老套餐用户续订和升级，撤回新体系页面，重新展示旧版套餐界面。普通用户端则显示售罄，新会员体系开放购买时间待定。

## 模型发布

### 5. 阿里云百炼上线 HappyOyster 1.0 并开启灰测

阿里云百炼上线 **HappyOyster 1.0** 并开启灰测，官方称其为“全球首个主动式实时交互世界模型”。该模型学习状态转移规律，支持用户在任意节点介入改变数字世界走向，具备世界探索与实时导演两大模式。

其中，世界探索模式支持 1 分钟连续实时交互；实时导演模式支持 3 分钟以上 480p 或 720p 实时画面和分支叙事。目前产品提供 Open API 和三端 SDK，可用于可玩世界、互动剧等沉浸式应用。

**相关链接：**

- https://mp.weixin.qq.com/s/kfbpyEZ9UY6-6vFudBK2tQ

### 6. 昆仑万维发布并开源 Matrix-Game 3.5

昆仑万维发布并开源 **Matrix-Game 3.5** 交互世界模型，5B 基础模型权重和代码已在 GitHub 与 Hugging Face 上线。

官方称该模型通过 Patch Memory 实现摄像机可控的 720p 视频生成，5B 模型在 720p 分辨率下具备 1 分钟记忆能力，并可达到单卡 20FPS 的实时生成性能。模型支持第一人称和第三人称模式，推理需至少 40GB 显存的 NVIDIA GPU。开源仓库显示，目前放出的是基础双向模型，蒸馏后的实时自回归模型尚未发布。

**相关链接：**

- https://huggingface.co/RiemannDynamics/Matrix-Game-3.5-Base
- https://mp.weixin.qq.com/s/LidvGePhOOoUY3KTor_w9g

### 7. 面壁智能发布 MiniCPM5-2B

面壁智能联合 OpenBMB 发布端侧模型 **MiniCPM5-2B**。官方称该模型在多项基准评测平均得分 54.26，在 4B 以下模型中性能居首，并原生支持混合思考与 **512K 超长上下文窗口**。

该模型目前暂未开源，后续将在 Hugging Face 等平台发布权重。众智 FlagOS 社区完成其 9 款芯片 Day0 适配，面壁智能也完成 AMD、Intel 等四家主流芯片厂商的端侧适配。同时公布的还有 UltraX 数据治理方案与智能体训练全流程技术，面壁智能与英特尔签署合作备忘录。

### 8. NVIDIA 与日本巨头合作建设物理 AI 基础设施

NVIDIA 正与日本政府支持的 Noetra 及多家制造、机器人企业推进物理 AI 合作。Noetra 已获软银、索尼、NEC、本田等 44 家公司及机构投资，计划建设搭载约 **2.75 万颗 NVIDIA Rubin GPU** 的 AI 计算基础设施。

丰田、富士通、发那科、安川电机、川崎重工等企业也分别在汽车、制造、机器人和智慧城市领域采用或评估 NVIDIA 的物理 AI 平台。
