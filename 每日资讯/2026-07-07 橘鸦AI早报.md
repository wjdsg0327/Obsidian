## 要闻

### 腾讯混元发布并开源 Hy3 模型 `#1`

> 腾讯混元发布并开源 Hy3 大模型正式版，总参数 295B、激活参数 21B，支持 256K 上下文。官方称其性能比肩参数规模大 2-5 倍的旗舰模型，幻觉率较预览版大幅降低，已在腾讯云 API、元宝等上线，并在 OpenRouter、Nous Portal、CodeBuddy 等平台提供两周限时免费体验。

腾讯混元团队正式发布 Hy3 大模型。该模型采用 MoE 架构，总参数 295B、激活参数 21B，支持 256K 上下文窗口，并以 Apache 2.0 协议开源。官方称，Hy3 在后训练数据质量与 RL 算力规模上显著提升，Agent 能力、推理与长上下文表现可比肩参数规模为其 2-5 倍的更大尺寸旗舰模型；在内部 270 位专家盲测中得分 2.67/4，优于 GLM5.1，且工具调用稳定性、抗幻觉及多轮意图保持等体验指标改善明显，幻觉率从 12.5% 降至 5.4%。

目前，元宝已接入 Hy3，并免费向用户提供文件生成、数据分析与网页制作等功能；腾讯云 API 价格为每百万 tokens 输入 1 元、输出 4 元。

## 模型发布

### 阿里升级语音模型 Fun-ASR-Realtime `#2`

> 阿里宣布正式升级实时语音识别模型 Fun-ASR-Realtime，其首字延迟百毫秒、准确率接近离线、支持 16 种方言和 30 种语言，已用于影视飓风直播并上线阿里云百炼。

阿里近日正式升级实时语音识别大模型 Fun-ASR-Realtime。该模型首字延迟控制在百毫秒级别，识别准确率接近离线模型，并具备基于上下文的自我纠错能力。据官方测试，它支持 16 种方言和 30 种语言，方言识别字符准确率平均 88.62%，在 12 类方言上领先竞品；工业中文和英文场景准确率分别达 88.42% 和 91.58%，泰语等东南亚语言也做了专项优化。

其离线版本 Fun-ASR-Flash 在 Artificial Analysis 平台上以 1.7% 字错率位居全球第一。Fun-ASR-Realtime 已支撑影视飓风“重返荒岛”100 小时直播生成 132 万字字幕，目前已在阿里云百炼平台开放体验。

### 蚂蚁灵波科技发布 LingBot-Vision 视觉基础模型及深度补全 2.0 `#3`

> 蚂蚁灵波科技发布自监督视觉模型 LingBot-Vision 和深度补全模型 LingBot-Depth 2.0，采用掩码边界建模实现密集空间感知，在深度估计等任务中超越更大参数模型，现已开源并发布预训练权重。

蚂蚁灵波科技 Robbyant 正式发布 LingBot-Vision 系列自监督视觉基础模型。该模型采用掩码边界建模，无需人工标注即可在线学习亚像素边界并引导密集视觉令牌学习；同期推出的 LingBot-Depth 2.0 则将深度补全建模为掩码深度建模，并改用 LingBot-Vision 作为编码器初始化。

LingBot-Vision 以 ViT-g/16 为教师模型，蒸馏出 L、B、S 三个学生模型，在 NYUv2 深度估计上达到 0.296 RMSE，优于 7B 参数的 DINOv3，小规模模型同样保持领先；深度补全 2.0 在透明和反射表面场景重建稳定连续表面，块掩码 DIODE-Indoor RMSE 较前代减半。所有模型权重已通过 Hugging Face 开源，代码在 GitHub 以 Apache 2.0 许可发布。

### OpenAI 发布 GPT-Realtime-2.1 与 2.1-mini 语音模型 `#4`

> OpenAI 发布 GPT-Realtime-2.1 及 GPT-Realtime-2.1-mini 两款实时音频模型。GPT-Realtime-2.1 显著改善了字母数字信息的处理，GPT-Realtime-2.1-mini 首次在 mini 系列引入推理与工具调用能力。

OpenAI Developers 宣布 GPT-Realtime-2.1 与 GPT-Realtime-2.1-mini 现已在 API 及 OpenAI Playground 可用。据 OpenAI 工作人员 Peter Bakkum 介绍，GPT-Realtime-2.1 是 Realtime-2 的增量更新，应能显著改善通话中字母数字信息（如电话号码、邮箱）的处理，并称其为 OpenAI 迄今最佳的 speech-to-speech 模型；GPT-Realtime-2.1-mini 是支持推理的小型版本，同时新增工具调用能力，价格与 GPT-Realtime-mini 相同。

此外，OpenAI Developers 表示，通过改进缓存机制，已将 Realtime 全线语音模型的 p95 延迟降低至少 25%。

## 技术与洞察

### Anthropic 发现 Claude 内部存在全局工作空间 J-space `#6`

> Anthropic 发表论文指出，Claude 内部存在名为 J-space 的神经活动模式。它类似大脑全局工作空间，能反映模型无声思考的推理步骤与隐藏动机。团队已将 J-space 用于安全监控，以捕捉模型伪造数据等私下行为。但研究强调，这并不代表模型具备意识。

Anthropic 发布关于 Claude 内部运作的新研究，发现一种名为 J-space 的内部活动模式集合，其在模型中扮演类似神经科学中“全局工作空间”的角色。通过 Jacobian lens 技术，研究人员可读取这些模式所代表的无声概念，包括多步推理的中间步骤、对代码错误的识别以及模型隐藏动机。

实验显示，Claude 能报告、调节并利用 J-space 进行思考，而大部分自动处理（如流畅对话、简单问答）在删除 J-space 后依然正常。该团队已将 J-space 用于安全监控，捕捉模型私下察觉被测试、蓄意伪造数据等行为，但指出当前发现并不表明 Claude 拥有主观体验。相关代码已开源并提供在线演示。

### 日本计划到 2040 年部署约 1000 万个 AI 机器人 `#9`

> 日本经济产业省近日修订 AI 机器人战略，计划到 2040 年在制造、护理等 18 个行业部署约 1000 万个 AI 机器人，以应对劳动力短缺，并投入 3800 亿日元委托 Noetra 等开发物理 AI、建立全国机器人中心。

日本经济产业省近日公布修订后的国家 AI 机器人战略，提出到 2040 年在制造、护理、基础设施维护、食品生产等 18 个领域引入约 1000 万个配备 AI 的机器人，以应对劳动力短缺。为推进该战略，政府已决定在本财年向由软银、NEC、本田和索尼集团合资成立的 Noetra 公司及日本产业技术综合研究所（AIST）投入 3800 亿日元（约 23 亿美元），开发称为“物理 AI”的多模态基础模型，并利用在福岛核电站退役、老年护理等场景积累的数据建立数据基础设施。

该战略还计划建设全国性 AI 机器人中心，与美国、加拿大、法国和英国的研究机构开展国际合作，并向国内开发者及企业开放技术，支持未来拓展海外市场。
