## 模型发布

### 1. InternLM 发布 Intern-S2-Preview-397B 多模态模型
- 亮点：面向科学智能与长程 Agent，支持最大 256K 文本和 64K 多模态推理长度，默认开启思维模式。
- 说明：官方称其在开源模型中具备领先通用推理性能，并强化了工具调用与时间序列推理。
- 相关链接：<https://huggingface.co/internlm/Intern-S2-Preview-397B> ｜ <https://chat.intern-ai.org.cn>

### 2. Google 推出 Gemma 4 更新，启用 FA4 并修补工具调用
- 亮点：在 NVIDIA Hopper GPU 上启用 Flash Attention 4，提升预填充吞吐量并降低首 token 时延。
- 说明：更新同时修补工具调用与聊天模板，相关量化版本需要重新下载。
- 相关链接：<https://huggingface.co/collections/google/gemma-4>

### 3. 中国气象局开源“风和”气象服务大语言模型
- 亮点：由公共气象服务中心联合智谱等研发，采用 GLM-4.5-Air 的 MoE 架构，上下文长度 128K。
- 说明：模型使用超 5000 万 tokens 气象语料和 49 万指令微调示例训练，权重已在多平台开放。
- 相关链接：<https://mp.weixin.qq.com/s/kc6JDKiosorc8tIWbLHqhQ> ｜ <https://github.com/PMSCCMA/FengHe> ｜ <https://huggingface.co/PMSCCMA/FengHe>

## 技术与洞察

### 1. OpenAI 提出 AI 时代投资回报计分卡：衡量每美元有用智能产出
- 核心：用有用工作量、成功任务成本、可靠性与算力回报四项指标衡量 AI 投入产出。
- 观点：单看 token 单价容易误导，前沿模型的总成本更取决于任务成功率。
- 相关链接：<https://openai.com/index/a-scorecard-for-the-ai-age/>

### 2. Anthropic 发文介绍 Claude Code 大规模代码迁移方法论
- 核心：用高度并行的 Agent 流程处理迁移，重点是修复“产生代码的流程”。
- 观点：强调规则、审查前置与自动化协作，而不是逐个修补代码。
- 相关链接：<https://claude.com/blog/ai-code-migration> ｜ <https://claude.com/blog/introducing-dynamic-workflows-in-claude-code>

### 3. Anthropic 副 CISO 发布 Agentic AI 风险管理指南
- 核心：用“不可信输入、可执行操作、失控影响范围、可观测性”四问评估 Agent 风险。
- 观点：通过最小权限、沙箱、网络白名单和遥测接入，将风险控制到可审计范围。
- 相关链接：<https://claude.com/blog/ciso-guide-to-agentic-ai>

### 4. AI Security Institute：开放模型网络能力差距缩至 4-7 个月
- 核心：开放权重模型与前沿闭源模型在网络能力上的差距继续缩小。
- 相关链接：<https://x.com/AISecurityInst/status/2078103148665667648>

### 5. 美团 LongCat 团队发布 LoHoSearch 搜索 Agent 基准测试
- 核心：基于 Wikipedia 知识图谱自动生成任务，评估搜索 Agent 的广度与深度能力。
- 相关链接：<https://arxiv.org/abs/2606.12837>
