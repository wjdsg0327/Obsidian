## 模型发布

### SeFi-Image 团队开源文本到图像模型，提供 1B 至 5B 参数规模与 Turbo 加速版本 `#2`

> SeFi-Image 团队发布了 SeFi-Image 系列开源文本转图像基础模型，采用语义优先扩散技术，并提供了 1B、2B 与 5B 等多种参数规模及 Turbo 加速版本，其代码与模型权重已在 GitHub 与 Hugging Face 发布。

SeFi-Image 团队近日正式开源了 SeFi-Image 系列文本到图像基础模型，该模型基于“语义优先扩散”架构，通过将语义结构与纹理细节的生成流分离，使语义潜变量先于纹理进行降噪，从而在纹理生成阶段提供更清晰的结构锚点。官方称，其 5B 版本仅使用约 125K A800 GPU 小时的训练资源，在 GenEval、LongTextBench、DPG-Bench 等多个基准测试中取得了具有竞争力或更强的结果。该系列提供了 Base、RL 和 Turbo 三类变体，覆盖 5B、2B 和 1B 参数规模，并同时发布了命令行及 Python API 形式的推理代码。

### 美团 LongCat 以 MIT 许可正式开源 LongCat-2.0 模型 `#3`

> 美团 LongCat 宣布已将 LongCat-2.0 以 MIT 许可全面开源，该模型为 1.6T 参数 MoE 架构，约 48B 参数激活，提供 GPU 与 NPU 部署方案和推理代码。

美团 LongCat 团队宣布 LongCat-2.0 模型正式以 MIT 许可证全面开源，并公开模型权重与推理代码。该模型采用 1.6T 参数的混合专家架构，每次推理激活约 480 亿参数，支持最长 100 万 token 上下文，定位为 Agent 原生模型，可直接集成 Claude Code、OpenClaw 和 Hermes Agent。部署方面，官方提供了 GPU 和 NPU 平台的推理支持，并透露已在国内大规模集群上完成验证。

## 产品应用

### Grok 以周共享额度取代产品分设日限，支持额外充值 `#4`

> Grok 更新文档向付费用户推行每周共享使用额度，替代过去按产品分设的每日限额。用户可跨 Chat、Imagine 等产品自由支配额度，额度每周重置，超限后还能购买额外点数或升级订阅。

Grok 更新文档为付费用户推出全新的每周使用限制系统，将原先 Chat、Imagine、Voice 和 Build 等产品各自的每日限额，统一为一个共享的每周额度池。该额度按操作消耗的计算资源扣减，聊天消耗低，而高质量视频生成或长编码任务消耗更高，用户可在任意产品间自由分配用量。达到周限制后，付费功能暂停，但免费层级的聊天和语音仍可继续使用；用户可选择购买额外使用额度、升级至更高订阅套餐或开启自动充值以立即恢复。

## 行业动态

### QTS 终止弗吉尼亚 Digital Gateway 数据中心项目 `#5`

> 据媒体报道，Blackstone 旗下数据中心运营商 QTS 日前终止了弗吉尼亚州 Digital Gateway 项目，并撤销相关上诉。该项目曾计划建设数十座数据中心，但因社区反对和法律挑战最终告终。

据路透社和《Prince William Times》报道，Blackstone 旗下数据中心运营商 QTS 于 7 月 2 日正式终止了其规划多年的 Digital Gateway 项目，并向弗吉尼亚州最高法院撤回了最后的上诉。该项目计划在盖恩斯维尔农村地区建设多达 37 个数据中心，预计投资数百亿美元，但因毗邻马纳萨斯国家战场公园引发社区强烈反对，并在法院诉讼中导致重新分区许可被撤销。在不利裁决接连作出后，QTS 最终决定放弃并承诺有序终止项目活动，同时强调弗吉尼亚州仍是其业务的重要组成部分。

### 报道称部分美国富裕家庭转向 AI 私立学校，年学费高达 7.5 万美元 `#6`

> 据华尔街日报报道，美国部分富裕家庭正转向结合人工智能的私立学校，例如 Alpha School。该校采用每日两小时 AI 辅导搭配项目制学习，最高年费约 7.5 万美元。

据华尔街日报报道称，美国部分富裕家庭正将子女从传统学校转出，转向以 AI 驱动的私立学校。以 Alpha School 为例，其教学模式为每日两小时 AI 个性化辅导加项目制工作坊，AI 平台可实时追踪学生参与度并调整课程。该校年学费最高达 7.5 万美元，去年新增 8 个校区，并计划今年秋季再开设近 20 个新地点。与此同时，报道指出传统学校对如何有效融合 AI 仍缺乏对策，多项研究表明学生使用 AI 作弊会导致长期成绩下滑，而此类高价私校的兴起也折射出 AI 时代日益加剧的教育鸿沟。

### 社区反对迫使 Fedora 叫停 AI 桌面倡议 `#7`

> Fedora 近期暂停了 AI Developer Desktop 计划，这是一款面向 AI 开发者的定制桌面系统，集成 GPU 加速与开发工具链。由于社区反对，官方已搁置该提案，相关工作转为独立项目继续探索。

日前，Fedora 委员会宣布暂停社区倡议流程，并关闭了 AI Developer Desktop 提案。该计划原拟打造一款基于 Atomic 架构的官方系统镜像，预配置 NVIDIA GPU 驱动、Goose CLI 和 Podman Desktop 等工具，以降低 AI 开发部署门槛。在征求意见阶段，大量社区成员提出反对，认为现行流程缺乏有效包容性，委员会因此决定不再通过旧机制推进此计划，并着手设计替代流程。目前，AI Developer Desktop 无法作为官方版本发布，但其非官方预览版已在社区仓库提供。

## 前瞻与传闻

### 报道称 NVIDIA Kyber 架构推迟至 2028 年，多款产品遭取消 `#8`

> 据 SemiAnalysis 报道，NVIDIA Kyber NVL144 架构延迟至 2028 年，替代方案 NVL72×2 和 4 芯片版 Rubin Ultra 也遭取消，Rubin Ultra 性能预期减半。NVIDIA 尚未就上述传闻置评。

SemiAnalysis 在近日发布的报告中称，NVIDIA 在 GTC 上展示仅三个月的 Kyber NVL144 机架架构因 PCB 中板制造难题，已推迟超过一年，预计要到 2028 年才能问世。同时，作为替代方案的 NVL72x2 背靠背架构已被取消，通过 CPO 连接多个 Oberon 机架的 NVL576 也可能延迟或限小批量，导致 Rubin Ultra 的纯铜 NVLink 扩展域受限；且 4 计算芯片版 Rubin Ultra 取消，仅保留性能约为原计划一半的 2 芯片版。此外，CPO NVSwitch 需等到 Feynman 才能使用，使 NVIDIA 目前缺乏扩大规模的有效方案，给竞争对手留下机会，而 NVIDIA 将销售更多 Oberon 机架来弥补。该报道迅速引发部分社区用户的质疑，指 SemiAnalysis 此前有发布看跌报告后推出 ETF 的先例，怀疑其动机；NVIDIA 尚未就上述传闻置评。
