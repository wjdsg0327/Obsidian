### [Anthropic上调Claude API速率限制并简化层级](https://x.com/ClaudeDevs/status/2072818299361263778)

> Anthropic近日宣布为所有用户提高 Claude Platform 上 API的调用频率上限，并简化了层级结构，不再基于API使用量分级。

Anthropic官方近期宣布已为所有用户提高Claude平台API的速率限制，并简化了层级，新的层级不再基于API使用量。最新的Sonnet和Haiku模型在最高层级提供5倍更高的速率限制，且各层级在Opus、Sonnet、Haiku模型上提供相同的每分钟请求数和令牌吞吐量，用户可根据任务选择模型而不受速率限制影响。层级提升自动进行，也可在Claude控制台手动申请。

### [社区报告 Claude Code 疑似跨会话缓存泄漏，Agent输出无关内容](https://github.com/anthropics/claude-code/issues/74066)

> 据用户报告，Claude Code 出现疑似跨会话缓存泄漏，Agent突然谈论Minecraft等无关内容。社区讨论认为可能是{缓存隔离缺陷|"缓存隔离缺陷"}或模型幻觉，Anthropic尚未公开回应。

据GitHub Issue #74066，发起者milesrichardson-edb 在使用 Claude Code 出现疑似跨会话缓存泄漏，Agent突然谈论Minecraft等无关内容。涉及Sonnet 5模型，首次响应超5分钟后发生。本地排查未找到对应文本，排除本地泄漏；移动端响应包含墙面尺寸等注入内容，模型自检注入。社区讨论分歧，部分认为是模型幻觉，将文件名线索拼凑为输出；另分析指缓存隔离缺陷，如key错误、buffer未清零，多租户GPU共享缓存增加风险。另有用户报告伪造通知、他人文件等类似经历。Anthropic曾解释一起事件为内部上下文错误呈现，但未公开确认此次报告，发起者已上报。

### [社区报告 Codex 推理token异常聚类或致复杂任务表现下降](https://github.com/openai/codex/issues/30364)

> 据多名开发者反映，GPT-5.5在Codex环境中的推理token数量出现异常聚类，频繁落在516及以518为步长的固定值上，导致部分复杂任务准确率下降。官方尚未回应，社区已提出临时规避方案。

近期，多位开发者在GitHub Issue和社交平台上报告，GPT-5.5在Codex产品中的推理输出token呈现反常的固定值聚类，主要集中在516以及1034、1552等以518为步长的值。聚合数据显示，gpt-5.5的精确516事件占比高达44%，远高于其他模型，且该比率在5月至6月间急剧上升，同时平均推理token强度明显下降，导致复杂任务错误率显著增加，已有大量用户复现此问题。对于原因，部分观点认为可能是系统提示机制导致的隐式截断，也有观点认为是批量推理优化所致，但OpenAI官方尚未对此公开说明。社区中已出现修改系统提示配置、使用特定工具或在提示词前端加入高强度推理指令等临时规避方法。

### [Anthropic 宣布自主开发药物，专家提醒 AI 仍难替代实验流程](https://www.ithome.com/0/972/606.htm)

> Anthropic 近期宣布启动自主药物研发，重点攻克传统药企因回报有限而长期忽视的罕见病。但有专家提醒，AI 仍无法替代漫长实验与监管审批流程。

Anthropic 于本月初举行活动，宣布启动自主药物研发项目，重点针对传统药企评定为商业回报有限而长期缺乏投入的“被忽视疾病”。该自主项目将聚焦新药研发早期阶段和临床前研究，但 Anthropic 尚未公布首批具体研究疾病，也未说明未来若发现候选药物将自主推进还是合作完成后续流程。与此同时，多位业内专家指出，AI 虽能参与药物研发多个环节并提升效率，但迄今尚无完全由 AI 设计的新药完成全部临床试验并获批，候选药物的毒性、安全性、药效及人体临床试验验证难以被 AI 明显压缩，因此该项目距离产生临床成果仍有较长距离。

### [报道称 Midjourney 动议要求好莱坞片厂全面披露内部 AI 使用](https://techcrunch.com/2026/07/04/midjourney-wants-hollywood-studios-to-reveal-the-details-of-their-ai-usage/)

> 据报道，AI公司Midjourney在与好莱坞制片厂的版权诉讼中提交动议，要求法官强制{Disney|"Disney"}、{Universal|"Universal"}和{Warner Bros.|"Warner Bros."}全面披露其自身的生成式AI使用情况，并指对方可能在内部同样使用未授权版权内容训练模型。

据TechCrunch报道，AI公司Midjourney在与Disney、Universal及Warner Bros.的版权诉讼中，近日提交动议要求法院撤销证据开示限制，以迫使三家制片厂披露其所有内部AI使用方式。此前法官裁定制片厂只需提供与“面向消费者”内容相关的生成式AI信息，但Midjourney认为这一限制“不公平”，并指制片厂可能在闭门开发同类图像生成AI，甚至同样利用未授权版权内容进行训练。Midjourney还要求对方提供其在该平台上使用过的所有提示词与生成结果，而非仅提供涉嫌侵权图像的提示词；制片厂首席律师则斥之为“钓鱼取证”，并重申诉讼目的仅是阻止未经授权使用其角色，并非阻碍AI技术发展。
