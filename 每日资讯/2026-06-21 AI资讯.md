## Hermes Agent 更新

- 仓库：`NousResearch/hermes-agent`
- 最新 release：`Hermes Agent v0.17.0 (v2026.6.19)`
  - GitHub release：`https://github.com/NousResearch/hermes-agent/releases/tag/v2026.6.19`
  - created_at：`2026-06-19T19:38:59Z`
  - published_at：`2026-06-19T19:39:06Z`
  - 官方 release note 称为 “The Reach Release”。
- 官方 release note 重点：
  - iMessage via Photon Spectrum：新增 iMessage platform plugin，基于 Photon managed line pool，不需要 Mac relay / BlueBubbles bridge。
  - Raft agent network：新增 bundled Raft platform adapter，让 Hermes 可通过 wake-channel bridge 接入 Raft，并强调 wake payload 只携带 metadata，不携带 message bodies。
  - Desktop app 增强：可重绑定快捷键、原生 OS notifications、subagent watch-windows、composer model selector、自动 RTL/bidi text direction、VS Code 风格 terminal pane、per-thread composer drafts、安装 VS Code Marketplace theme 等。
  - Background / async subagents：`delegate_task(background=true)` 可后台派发子代理并立即返回 handle，完成后将完整结果作为新 turn 回到对话。
  - `image_generate` 支持 image-to-image / 图片编辑。
  - Automation Blueprints：通过蓝图配置自动化任务，减少手写 cron 的需求。
  - xAI Grok subscription 下可触达 Cursor Composer model（release note 中提到 `grok-composer-2.5-fast`）。
- 最新 commits API 显示，release 后仍有 2026-06-21 UTC 后续提交：
  - `2213ea9fa73ab06cf667c1bfb1e99c8de3541589`：`test(whatsapp): cover read-only bridge dir mirror; add author map`
