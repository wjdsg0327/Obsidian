### 1. 过滤：保留满足条件的元素（Filter）

**场景：过滤出所有偶数**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	numbers := []int{1, 2, 3, 4, 5, 6}
	evens := lo.Filter(numbers, func(x int, _ int) bool {
		return x%2 == 0
	})
	fmt.Println(evens)
	// evens: [2, 4, 6] （优雅，永不过时）
}
```

### 2. 映射：把每个元素变换成新值（Map）

**场景：把每个数字翻倍**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	numbers := []int{1, 2, 3, 4, 5, 6}
	doubled := lo.Map(numbers, func(x int, _ int) int { return x * 2 })
	fmt.Println(doubled)
}
```

### 3. 组合：先过滤再映射（Filter + Map）

**场景：过滤 + 翻倍，组合拳**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	numbers := []int{1, 2, 3, 4, 5, 6}
	result := lo.Map(
		lo.Filter(numbers, func(x int, _ int) bool {
			return x%2 == 0
		}),
		func(x int, _ int) int {
			return x * 2
		},
	)
	// result: [4, 8, 12]
	fmt.Println(result)
}
```

### 4. Map 取值与统计（ValueOr / CountValues）

**场景：安全地获取 map 中的值 + 统计元素出现次数**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	myMap := map[string]int{"a": 1, "b": 2}
	names := []string{"apple", "banana", "apple", "orange", "banana", "apple"}

	// 1) 优雅取值（带默认值）
	value := lo.ValueOr(myMap, "c", 0)
	fmt.Println(value)
	// 在 map 中找键 "c"，找不到就返回默认值 0

	// 2) 统计元素出现次数
	exists := lo.CountValues(names)
	fmt.Println(exists)
}
```

### 5. 切片转 Map：快速索引（KeyBy）

**场景：结构体列表按 ID 建立索引**

```go
type User struct {
	ID   int
	Name string
}

func main() {
	users := []User{{ID: 1, Name: "Alice"}, {ID: 2, Name: "Bob"}}

	// 以 ID 为 key 生成 map
	userMap := lo.KeyBy(users, func(u User) int {
		return u.ID
	})

	fmt.Printf("%+v\\n", userMap[1]) // {ID: 1, Name: Alice}
}
```

### 6. 去重：删除重复元素（Uniq）

**场景：对切片去重**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	names := []string{"Samuel", "John", "Samuel", "Erik"}
	uniqueNames := lo.Uniq(names)
	fmt.Println(uniqueNames) // [Samuel John Erik]
}
```

### 7. 分组：按条件聚合（GroupBy）

**场景：按奇偶分组**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	numbers := []int{1, 2, 3, 4, 5, 6, 7, 8}

	groups := lo.GroupBy(numbers, func(x int) string {
		if x%2 == 0 {
			return "even"
		}
		return "odd"
	})

	fmt.Println(groups) // map[even:[2 4 6 8] odd:[1 3 5 7]]
}
```

### 8. 提取字段：从结构体切片抽取某一列（Map）

**场景：把所有用户的 ID 提取成切片**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

type User struct {
	ID   int
	Name string
}

func main() {
	users := []User{{ID: 1, Name: "Alice"}, {ID: 2, Name: "Bob"}}

	ids := lo.Map(users, func(u User, _ int) int {
		return u.ID
	})

	fmt.Println(ids) // [1, 2]
}
```

### 9. 扁平化：把二维切片拍平（Flatten）

**场景：二维切片 -> 一维切片**

```go
package main

import (
	"fmt"
	"github.com/samber/lo"
)

func main() {
	nested := [][]int{{1, 2}, {3, 4}, {5}}
	flat := lo.Flatten(nested)
	fmt.Println(flat) // [1, 2, 3, 4, 5]
}
```
