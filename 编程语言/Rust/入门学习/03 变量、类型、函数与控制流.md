# 03 变量、类型、函数与控制流

这一节先把 Rust 的基本语法跑通。重点不是背，而是知道“Rust 代码长什么样”。

## 变量默认不可变

Rust 里变量默认不能修改：

```rust
fn main() {
    let x = 5;
    // x = 6; // 报错
    println!("x = {}", x);
}
```

如果要修改，需要加 `mut`：

```rust
fn main() {
    let mut x = 5;
    x = 6;
    println!("x = {}", x);
}
```

这体现了 Rust 的风格：默认安全，明确声明可变。

## 变量遮蔽 shadowing

Rust 允许用同名变量重新绑定：

```rust
fn main() {
    let x = 5;
    let x = x + 1;
    let x = x * 2;
    println!("x = {}", x); // 12
}
```

这不是修改原变量，而是创建了一个新的 `x`。

可以改变类型：

```rust
fn main() {
    let spaces = "   ";
    let spaces = spaces.len();
    println!("空格数量: {}", spaces);
}
```

## 常见基础类型

### 整数

```rust
let a: i32 = -10;
let b: u32 = 10;
let c: i64 = 100000;
```

- `i` 表示 signed，有正负
- `u` 表示 unsigned，只能非负
- 常用默认整数类型是 `i32`

### 浮点数

```rust
let x: f64 = 3.14;
let y: f32 = 2.5;
```

默认浮点类型是 `f64`。

### 布尔

```rust
let ok: bool = true;
let failed = false;
```

### 字符

```rust
let c = '中';
let emoji = '🌸';
```

Rust 的 `char` 是 Unicode 字符，不只是 ASCII。

### 元组 tuple

```rust
let user = ("老王", 18, true);
println!("名字: {}", user.0);
println!("年龄: {}", user.1);
```

### 数组 array

```rust
let nums = [1, 2, 3, 4, 5];
println!("第一个: {}", nums[0]);
```

数组长度固定。

## 函数

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn main() {
    let result = add(3, 4);
    println!("result = {}", result);
}
```

注意：

- 参数必须写类型
- 返回值用 `->`
- 最后一行 `a + b` 没有分号，表示返回表达式

如果加分号：

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b;
}
```

会报错，因为 `a + b;` 是语句，不返回值。

## 表达式和语句

Rust 很多东西是表达式，可以产生值。

```rust
let x = {
    let a = 1;
    let b = 2;
    a + b
};

println!("x = {}", x); // 3
```

## if 表达式

```rust
fn main() {
    let age = 20;

    if age >= 18 {
        println!("成年人");
    } else {
        println!("未成年人");
    }
}
```

`if` 也能返回值：

```rust
let score = 80;
let level = if score >= 60 { "及格" } else { "不及格" };
println!("{}", level);
```

注意两个分支返回类型必须一样。

## loop 循环

```rust
fn main() {
    let mut count = 0;

    loop {
        count += 1;
        if count == 3 {
            break;
        }
    }
}
```

`loop` 可以返回值：

```rust
let mut count = 0;
let result = loop {
    count += 1;
    if count == 10 {
        break count * 2;
    }
};
println!("result = {}", result);
```

## while 循环

```rust
let mut n = 3;
while n > 0 {
    println!("{}", n);
    n -= 1;
}
```

## for 循环

```rust
let nums = [10, 20, 30];
for n in nums {
    println!("{}", n);
}
```

范围：

```rust
for i in 1..5 {
    println!("{}", i); // 1,2,3,4
}

for i in 1..=5 {
    println!("{}", i); // 1,2,3,4,5
}
```

## 本节练习

写一个程序：

- 定义函数 `is_even(n: i32) -> bool`
- 判断 1 到 10 哪些是偶数
- 用 `for` 打印结果

参考：

```rust
fn is_even(n: i32) -> bool {
    n % 2 == 0
}

fn main() {
    for i in 1..=10 {
        if is_even(i) {
            println!("{} 是偶数", i);
        } else {
            println!("{} 是奇数", i);
        }
    }
}
```

## 本节小结

这节要记住：

- `let` 默认不可变
- `mut` 表示可变
- 函数参数和返回值要写类型
- Rust 很多结构是表达式
- `for i in 1..=10` 很常用
