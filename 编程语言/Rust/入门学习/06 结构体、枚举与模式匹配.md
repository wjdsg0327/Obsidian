# 06 结构体、枚举与模式匹配

Rust 里组织数据主要靠：

- `struct`：结构体
- `enum`：枚举
- `match`：模式匹配

这三者用好了，代码会非常清晰。

## struct 结构体

定义一个用户：

```rust
struct User {
    name: String,
    age: u32,
    active: bool,
}

fn main() {
    let user = User {
        name: String::from("老王"),
        age: 18,
        active: true,
    };

    println!("{}", user.name);
}
```

## 可变结构体

```rust
let mut user = User {
    name: String::from("老王"),
    age: 18,
    active: true,
};

user.age = 19;
```

注意：整个变量要 `mut`，Rust 不支持只让某个字段可变。

## 字段初始化简写

```rust
fn build_user(name: String, age: u32) -> User {
    User {
        name,
        age,
        active: true,
    }
}
```

当变量名和字段名一样，可以简写。

## 结构体更新语法

```rust
let user2 = User {
    name: String::from("小张"),
    ..user1
};
```

注意：如果 `user1` 里有 `String` 这类非 Copy 字段被移动，`user1` 之后可能不能完整使用。

## 给结构体定义方法

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }

    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
}

fn main() {
    let rect = Rectangle { width: 30, height: 50 };
    println!("面积: {}", rect.area());
}
```

`&self` 表示借用当前对象。

## 关联函数

不带 `self` 的函数叫关联函数，常用于构造：

```rust
impl Rectangle {
    fn square(size: u32) -> Rectangle {
        Rectangle {
            width: size,
            height: size,
        }
    }
}

let sq = Rectangle::square(10);
```

## enum 枚举

枚举表示一个值可能是几种情况之一。

```rust
enum Direction {
    Up,
    Down,
    Left,
    Right,
}
```

使用：

```rust
let d = Direction::Up;
```

## 枚举可以携带数据

```rust
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
```

这比很多语言的 enum 强大很多。

## match 模式匹配

```rust
fn handle_direction(d: Direction) {
    match d {
        Direction::Up => println!("向上"),
        Direction::Down => println!("向下"),
        Direction::Left => println!("向左"),
        Direction::Right => println!("向右"),
    }
}
```

`match` 必须穷尽所有可能。

## match 返回值

```rust
let score = 85;
let level = match score {
    90..=100 => "优秀",
    60..=89 => "及格",
    _ => "不及格",
};

println!("{}", level);
```

`_` 表示其它情况。

## if let 简化写法

当只关心一种情况时：

```rust
let value = Some(3);

if let Some(x) = value {
    println!("值是 {}", x);
}
```

比完整 `match` 更简洁。

## Option 枚举

Rust 没有 null，而是用 `Option<T>`：

```rust
enum Option<T> {
    Some(T),
    None,
}
```

例子：

```rust
fn find_user(id: u32) -> Option<String> {
    if id == 1 {
        Some(String::from("老王"))
    } else {
        None
    }
}
```

使用：

```rust
match find_user(1) {
    Some(name) => println!("找到: {}", name),
    None => println!("没找到"),
}
```

## 本节练习

定义一个任务状态：

```rust
enum TaskStatus {
    Todo,
    Doing,
    Done,
}

struct Task {
    title: String,
    status: TaskStatus,
}

fn print_task(task: &Task) {
    let text = match task.status {
        TaskStatus::Todo => "待办",
        TaskStatus::Doing => "进行中",
        TaskStatus::Done => "已完成",
    };

    println!("{} - {}", task.title, text);
}
```

## 本节小结

- `struct` 组织多个字段
- `impl` 给结构体定义方法
- `enum` 表示多种可能
- `match` 强制处理所有情况
- `Option<T>` 是 Rust 替代 null 的方式
