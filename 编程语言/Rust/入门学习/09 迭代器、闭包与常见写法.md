# 09 迭代器、闭包与常见写法

Rust 里很多代码会用迭代器和闭包，写熟以后会很简洁。

## 闭包 closure

闭包就是匿名函数：

```rust
let add = |a, b| a + b;
println!("{}", add(1, 2));
```

带类型也可以：

```rust
let add = |a: i32, b: i32| -> i32 { a + b };
```

## 闭包可以捕获环境

```rust
let x = 10;
let add_x = |n| n + x;
println!("{}", add_x(5));
```

## 迭代器 iterator

`Vec` 可以转成迭代器：

```rust
let nums = vec![1, 2, 3];
let iter = nums.iter();
```

常见三种：

```rust
nums.iter();      // 借用元素：&T
nums.iter_mut();  // 可变借用：&mut T
nums.into_iter(); // 拿走所有权：T
```

## map

把每个元素转换成另一个值：

```rust
let nums = vec![1, 2, 3];
let doubled: Vec<i32> = nums.iter().map(|n| n * 2).collect();
println!("{:?}", doubled);
```

## filter

过滤元素：

```rust
let nums = vec![1, 2, 3, 4, 5];
let evens: Vec<i32> = nums.into_iter().filter(|n| n % 2 == 0).collect();
println!("{:?}", evens);
```

## collect

把迭代器收集成集合：

```rust
let nums: Vec<i32> = (1..=5).collect();
println!("{:?}", nums);
```

有时需要标注类型：

```rust
let nums = (1..=5).collect::<Vec<i32>>();
```

## sum

```rust
let total: i32 = (1..=100).sum();
println!("{}", total);
```

## any / all

```rust
let nums = vec![1, 2, 3];
let has_even = nums.iter().any(|n| n % 2 == 0);
let all_positive = nums.iter().all(|n| *n > 0);
```

## find

```rust
let nums = vec![1, 2, 3, 4];
let found = nums.iter().find(|n| **n > 2);
println!("{:?}", found);
```

这里 `n` 类型是 `&&i32`，所以用了 `**n`。

## enumerate

拿到下标：

```rust
let names = vec!["老王", "小张"];
for (index, name) in names.iter().enumerate() {
    println!("{}: {}", index, name);
}
```

## 常见链式写法

```rust
let result: Vec<String> = vec![" rust ", "", " cargo "]
    .into_iter()
    .map(|s| s.trim())
    .filter(|s| !s.is_empty())
    .map(|s| s.to_uppercase())
    .collect();

println!("{:?}", result);
```

含义：

1. 去掉前后空格
2. 过滤空字符串
3. 转大写
4. 收集成 Vec

## match + iterator 的实际例子

把字符串数字转成整数，并忽略非法项：

```rust
let items = vec!["1", "2", "x", "4"];

let nums: Vec<i32> = items
    .into_iter()
    .filter_map(|s| s.parse::<i32>().ok())
    .collect();

println!("{:?}", nums);
```

`parse::<i32>()` 返回 `Result<i32, _>`。

`.ok()` 把 `Result` 转成 `Option`：

- `Ok(v)` → `Some(v)`
- `Err(_)` → `None`

`filter_map` 会自动丢掉 `None`。

## 什么时候用 for，什么时候用 iterator

用 `for`：

- 新手阶段
- 逻辑比较复杂
- 需要多个步骤和调试

用 iterator：

- 转换、过滤、求和这类数据流水线
- 代码能明显更清楚时

不要为了炫技硬写链式调用。

## 本节练习

把下面数组里大于 3 的数字乘以 10：

```rust
fn main() {
    let nums = vec![1, 2, 3, 4, 5];

    let result: Vec<i32> = nums
        .into_iter()
        .filter(|n| *n > 3)
        .map(|n| n * 10)
        .collect();

    println!("{:?}", result);
}
```

## 本节小结

- 闭包写法：`|x| x + 1`
- `iter()` 借用
- `iter_mut()` 可变借用
- `into_iter()` 拿走所有权
- `map/filter/collect` 是最常用组合
