# 10 实战：写一个命令行待办工具

这一节做一个小项目：命令行待办工具 `todo_cli`。

功能：

- 添加任务
- 列出任务
- 删除任务
- 数据保存到本地 JSON 文件

会用到：

- `clap` 解析命令行参数
- `serde` 处理 JSON
- `Result` 错误处理
- `Vec` 存任务列表
- 文件读写

## 创建项目

```bash
cargo new todo_cli
cd todo_cli
```

修改 `Cargo.toml`：

```toml
[package]
name = "todo_cli"
version = "0.1.0"
edition = "2021"

[dependencies]
anyhow = "1"
clap = { version = "4", features = ["derive"] }
serde = { version = "1", features = ["derive"] }
serde_json = "1"
```

## 完整代码

`src/main.rs`：

```rust
use anyhow::Result;
use clap::{Parser, Subcommand};
use serde::{Deserialize, Serialize};
use std::fs;
use std::path::Path;

const DB_PATH: &str = "todos.json";

#[derive(Debug, Serialize, Deserialize)]
struct Todo {
    id: u32,
    title: String,
    done: bool,
}

#[derive(Parser)]
#[command(name = "todo")]
#[command(about = "一个 Rust 命令行待办工具")]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    Add { title: String },
    List,
    Done { id: u32 },
    Remove { id: u32 },
}

fn load_todos() -> Result<Vec<Todo>> {
    if !Path::new(DB_PATH).exists() {
        return Ok(Vec::new());
    }

    let content = fs::read_to_string(DB_PATH)?;
    let todos = serde_json::from_str(&content)?;
    Ok(todos)
}

fn save_todos(todos: &[Todo]) -> Result<()> {
    let content = serde_json::to_string_pretty(todos)?;
    fs::write(DB_PATH, content)?;
    Ok(())
}

fn next_id(todos: &[Todo]) -> u32 {
    todos.iter().map(|todo| todo.id).max().unwrap_or(0) + 1
}

fn main() -> Result<()> {
    let cli = Cli::parse();
    let mut todos = load_todos()?;

    match cli.command {
        Commands::Add { title } => {
            let todo = Todo {
                id: next_id(&todos),
                title,
                done: false,
            };
            todos.push(todo);
            save_todos(&todos)?;
            println!("任务已添加");
        }
        Commands::List => {
            if todos.is_empty() {
                println!("暂无任务");
            } else {
                for todo in &todos {
                    let mark = if todo.done { "x" } else { " " };
                    println!("{} [{}] {}", todo.id, mark, todo.title);
                }
            }
        }
        Commands::Done { id } => {
            match todos.iter_mut().find(|todo| todo.id == id) {
                Some(todo) => {
                    todo.done = true;
                    save_todos(&todos)?;
                    println!("任务已完成");
                }
                None => println!("没有找到 id={} 的任务", id),
            }
        }
        Commands::Remove { id } => {
            let old_len = todos.len();
            todos.retain(|todo| todo.id != id);

            if todos.len() == old_len {
                println!("没有找到 id={} 的任务", id);
            } else {
                save_todos(&todos)?;
                println!("任务已删除");
            }
        }
    }

    Ok(())
}
```

## 运行

添加任务：

```bash
cargo run -- add "学习 Rust 所有权"
cargo run -- add "写一个 CLI 工具"
```

列出任务：

```bash
cargo run -- list
```

完成任务：

```bash
cargo run -- done 1
```

删除任务：

```bash
cargo run -- remove 2
```

## 代码拆解

### clap 解析命令

```rust
#[derive(Parser)]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}
```

表示命令行会有子命令。

```rust
enum Commands {
    Add { title: String },
    List,
    Done { id: u32 },
    Remove { id: u32 },
}
```

对应：

```bash
todo add "任务"
todo list
todo done 1
todo remove 1
```

### serde 保存 JSON

```rust
#[derive(Serialize, Deserialize)]
struct Todo { ... }
```

让结构体可以转 JSON，也可以从 JSON 读回来。

### retain 删除元素

```rust
todos.retain(|todo| todo.id != id);
```

保留所有 id 不等于目标 id 的任务，相当于删除目标任务。
