# C++ 学习资源指南

- **LearnCpp.com** - https://www.learncpp.com/

  - 交互式浏览器学习，快速入门

- **Udemy "Beginning C++"** - https://www.udemy.com/course/beginning-c-plus-plus-programming/
  - 流行的结构化课程，包含项目

### 官方资源

## 好玩的库

### 游戏开发
- **raylib** - https://www.raylib.com/
  - 简单、初学者友好的游戏编程库
  - 快速制作2D/3D游戏，设置简单

- **SFML** - https://www.sfml-dev.org/
  - 多媒体库：游戏、音频、图形、网络
  - API清晰，文档完善，社区活跃

- **Box2D** - https://box2d.org/
  - 2D物理引擎
  - 学习物理模拟，构建物理游戏

### 创意编程
- **OpenFrameworks** - https://openframeworks.cc/
  - C++创意编程工具包
  - 受艺术家和设计师欢迎，适合交互式装置和视觉效果

- **Cinder** - https://libcinder.org/
  - 专业品质的创意编程
  - 美丽的图形、动画和音频功能

### GUI开发
- **Dear ImGui** - https://github.com/ocornut/imgui
  - 即时模式GUI
  - 适合工具、调试UI和编辑器
  - 在游戏开发和实时应用中非常流行

- **Qt Framework** - https://www.qt.io/
  - 功能齐全的跨平台GUI框架
  - 桌面应用开发的行业标准

### 音频
- **JUCE** - https://juce.com/
  - 构建音频插件、合成器和音乐应用
  - 音频软件的行业标准

### 库大全
- **awesome-cpp** - https://github.com/fffaraz/awesome-cpp
  - 大型精选C++库列表，按类别组织
  - 涵盖AI、图形、网络、数学等

## 高阶玩法

### 现代C++特性
- **Modern C++ Features** - https://github.com/AnthonyCalandra/modern-cpp-features
  - C++11/14/17/20/23所有主要特性总结和示例
  - 保持最新的必备参考

- **C++23 参考** - https://en.cppreference.com/w/cpp/23
  - 最新C++23特性完整参考
  - 包括std::expected、std::print、ranges改进

### 设计模式
- **Refactoring Guru (C++)** - https://refactoring.guru/design-patterns/cpp
  - 所有主要设计模式（创建型、结构型、行为型）
  - C++实现和UML图

### 最佳实践
- **C++ Core Guidelines** - https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines
  - Bjarne Stroustrup和Herb Sutter的官方指南
  - 编写正确、现代C++的权威指南

### 必读书籍
- **"Effective Modern C++"** - Scott Meyers
  - 42种改进C++11/14代码的具体方法
  - 涵盖智能指针、移动语义、lambda、并发

- **"C++ Templates: Complete Guide" 第2版**
  - 模板的权威参考，从基础到高级元编程和SFINAE

- **"C++ Concurrency in Action"** - Anthony Williams
  - 多线程和并发编程的全面指南

### 博客和YouTube
- **Fluent C++ 博客** - https://www.fluentcpp.com/
  - 表达性强、干净的C++代码技术
  - 模板、CRTP、基于策略的设计等

- **C++ Weekly (YouTube)** - https://www.youtube.com/c/cppweekly
  - Jason Turner每周现代C++技巧、最佳实践和编译器探索

### 课程
- **Udemy Advanced C++** - https://www.udemy.com/course/beg-advanced-cpp-programming/
  - 移动语义、智能指针、模板、多线程

## 快速开始建议

### 绝对初学者
1. 从 LearnCpp.com 开始
2. 跟随 roadmap.sh/cpp 路线图
3. 使用 raylib 或 SFML 练习小项目

### 中级程序员
1. 阅读 "Effective Modern C++" (Scott Meyers)
2. 学习 C++ Core Guidelines
3. 探索 awesome-cpp 寻找有趣的库

### OpenCV + C++
1. 使用官方入门指南设置环境
2. 跟随 LearnOpenCV 教程
3. 浏览 GitHub 项目：github.com/topics/opencv-cpp
