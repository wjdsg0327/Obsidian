# OpenCV 配套示例脚本

这个目录收集了从 `docs/` 教程中拆分出来的可直接运行示例脚本。

如果运行脚本时出现 `ModuleNotFoundError: No module named 'cv2'`，先安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方式

```bash
python examples/ch00_p01.py
```

大部分脚本会弹出 `cv2.imshow()` 窗口，按任意键关闭；视频类脚本一般按 `q` 退出。
