# C++：工程模板与编译

本篇记录 Windows API C++ 项目的常见编译方式、工程结构和链接库处理。

## Visual Studio 项目建议

- 项目类型：Windows Desktop Application / Empty Project。
- 字符集：Use Unicode Character Set。
- C++ 标准：C++17 或更新。
- 常用宏：`UNICODE`、`_UNICODE`、`WIN32_LEAN_AND_MEAN`。
- 头文件：多数基础 API 直接包含 `<windows.h>`。

## MSVC 命令行编译

```bat
cl /EHsc /DUNICODE /D_UNICODE main.cpp user32.lib gdi32.lib
```

窗口子系统：

```bat
cl /EHsc /DUNICODE /D_UNICODE main.cpp /link /SUBSYSTEM:WINDOWS user32.lib gdi32.lib
```

## CMake 模板

```cmake
cmake_minimum_required(VERSION 3.20)
project(WinApiDemo LANGUAGES CXX)
set(CMAKE_CXX_STANDARD 17)
add_executable(WinApiDemo WIN32 main.cpp)
target_compile_definitions(WinApiDemo PRIVATE UNICODE _UNICODE WIN32_LEAN_AND_MEAN)
target_link_libraries(WinApiDemo PRIVATE user32 gdi32 shell32 ole32 advapi32)
```

## RAII 封装 HANDLE

```cpp
#include <windows.h>
#include <utility>

class unique_handle {
    HANDLE h_ = nullptr;
public:
    unique_handle() = default;
    explicit unique_handle(HANDLE h) : h_(h) {}
    ~unique_handle() { reset(); }
    unique_handle(const unique_handle&) = delete;
    unique_handle& operator=(const unique_handle&) = delete;
    unique_handle(unique_handle&& other) noexcept : h_(std::exchange(other.h_, nullptr)) {}
    unique_handle& operator=(unique_handle&& other) noexcept {
        if (this != &other) { reset(); h_ = std::exchange(other.h_, nullptr); }
        return *this;
    }
    void reset(HANDLE h = nullptr) {
        if (h_ && h_ != INVALID_HANDLE_VALUE) CloseHandle(h_);
        h_ = h;
    }
    HANDLE get() const { return h_; }
    explicit operator bool() const { return h_ && h_ != INVALID_HANDLE_VALUE; }
};
```

## 常见链接库

- `Kernel32.lib`：文件、进程、线程、内存、同步。
- `User32.lib`：窗口、消息、控件基础、剪贴板。
- `Gdi32.lib`：GDI 绘图。
- `Advapi32.lib`：注册表、服务、安全、ETW 控制。
- `Ole32.lib`：COM、OLE。
- `Shell32.lib`：Shell API。
- `Ws2_32.lib`：Winsock。
- `Winhttp.lib`：WinHTTP。
- `Crypt32.lib` / `Bcrypt.lib`：证书、加密。
