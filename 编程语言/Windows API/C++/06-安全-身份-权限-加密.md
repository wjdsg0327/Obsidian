# C++：安全、身份、权限、加密

Token、SID、ACL、权限提升、认证授权、CryptoAPI/CNG、证书、DPAPI。

## 常用 API 速查

| API | 用途 | 库/模块 |
|---|---|---|
| `OpenProcessToken` | 打开进程令牌 | Advapi32.lib |
| `LookupPrivilegeValue` | 查找权限 LUID | Advapi32.lib |
| `AdjustTokenPrivileges` | 启用/禁用权限 | Advapi32.lib |
| `GetTokenInformation` | 查询 Token | Advapi32.lib |
| `ConvertSidToStringSid` | SID 转字符串 | Advapi32.lib |
| `InitializeSecurityDescriptor` | 安全描述符 | Advapi32.lib |
| `AccessCheck` | 访问检查 | Advapi32.lib |
| `CryptProtectData/CryptUnprotectData` | DPAPI 加解密 | Crypt32.lib |
| `BCryptOpenAlgorithmProvider` | CNG 算法 | Bcrypt.lib |
| `CertOpenSystemStore/CertEnumCertificatesInStore` | 证书存储 | Crypt32.lib |

## 使用要点

- 查看每个 API 文档中的 **Header / Library / DLL / Minimum supported client**。
- C++ 默认建议走 Unicode 版本，避免 ANSI 代码页问题。
- 明确资源释放函数：`CloseHandle`、`LocalFree`、`CoTaskMemFree`、`DeleteObject`、`Release` 等不可混用。
- 失败处理：Win32 多数用 `GetLastError()`，COM 多数用 `HRESULT`。
