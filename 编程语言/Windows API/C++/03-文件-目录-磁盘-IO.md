# C++：文件、目录、磁盘、I/O

文件句柄、同步/异步 I/O、目录、卷、路径、内存映射文件。

## 常用 API 速查

| API | 用途 | 库/模块 |
|---|---|---|
| `CreateFile` | 打开/创建文件、设备、管道 | Kernel32.lib |
| `ReadFile/WriteFile` | 读写文件或设备 | Kernel32.lib |
| `CloseHandle` | 关闭句柄 | Kernel32.lib |
| `GetFileAttributesEx` | 读取文件属性 | Kernel32.lib |
| `FindFirstFile/FindNextFile` | 遍历目录 | Kernel32.lib |
| `CreateDirectory/RemoveDirectory` | 创建/删除目录 | Kernel32.lib |
| `DeleteFile/MoveFileEx/CopyFile` | 文件操作 | Kernel32.lib |
| `GetFileSizeEx/GetFileTime` | 大小和时间 | Kernel32.lib |
| `CreateFileMapping/MapViewOfFile` | 内存映射文件 | Kernel32.lib |
| `DeviceIoControl` | 设备控制与磁盘查询 | Kernel32.lib |
| `GetDiskFreeSpaceEx` | 磁盘空间 | Kernel32.lib |
| `GetVolumeInformation` | 卷信息 | Kernel32.lib |

## 使用要点

- 查看每个 API 文档中的 **Header / Library / DLL / Minimum supported client**。
- C++ 默认建议走 Unicode 版本，避免 ANSI 代码页问题。
- 明确资源释放函数：`CloseHandle`、`LocalFree`、`CoTaskMemFree`、`DeleteObject`、`Release` 等不可混用。
- 失败处理：Win32 多数用 `GetLastError()`，COM 多数用 `HRESULT`。

## 案例

```cpp
#include <windows.h>
#include <iostream>

int wmain() {
    HANDLE h = CreateFileW(L"demo.txt", GENERIC_WRITE, 0, nullptr, CREATE_ALWAYS,
                           FILE_ATTRIBUTE_NORMAL, nullptr);
    if (h == INVALID_HANDLE_VALUE) {
        std::wcerr << L"CreateFile failed: " << GetLastError() << L"
";
        return 1;
    }
    const char text[] = "hello windows api
";
    DWORD written = 0;
    WriteFile(h, text, sizeof(text)-1, &written, nullptr);
    CloseHandle(h);
}
```
