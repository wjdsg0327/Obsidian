# C++：图形、音视频、多媒体

GDI/GDI+、Direct2D/DirectWrite、Direct3D、WIC、Media Foundation、Core Audio。

## 常用 API 速查

| API | 用途 | 库/模块 |
|---|---|---|
| `BeginPaint/EndPaint` | 窗口绘制 | User32/Gdi32 |
| `GetDC/ReleaseDC` | 设备上下文 | User32.lib |
| `CreatePen/CreateBrush/DeleteObject` | GDI 对象 | Gdi32.lib |
| `BitBlt/StretchBlt` | 位图拷贝 | Gdi32.lib |
| `GdiplusStartup` | GDI+ 初始化 | Gdiplus.lib |
| `D2D1CreateFactory` | Direct2D 工厂 | D2d1.lib |
| `DWriteCreateFactory` | DirectWrite | Dwrite.lib |
| `CoCreateInstance(IMF*)` | Media Foundation | Mfplat/Mfreadwrite |
| `CoCreateInstance(MMDeviceEnumerator)` | Core Audio | Ole32.lib |
| `WICCreateImagingFactory` | WIC 图像编解码 | Windowscodecs.lib |

## 使用要点

- 查看每个 API 文档中的 **Header / Library / DLL / Minimum supported client**。
- C++ 默认建议走 Unicode 版本，避免 ANSI 代码页问题。
- 明确资源释放函数：`CloseHandle`、`LocalFree`、`CoTaskMemFree`、`DeleteObject`、`Release` 等不可混用。
- 失败处理：Win32 多数用 `GetLastError()`，COM 多数用 `HRESULT`。
