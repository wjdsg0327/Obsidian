# C++：窗口、消息、控件

窗口类、消息循环、窗口过程、菜单、资源、通用控件、DWM、高 DPI。

## 常用 API 速查

| API | 用途 | 库/模块 |
|---|---|---|
| `RegisterClassEx` | 注册窗口类 | User32.lib |
| `CreateWindowEx` | 创建窗口 | User32.lib |
| `ShowWindow` | 显示窗口 | User32.lib |
| `UpdateWindow` | 刷新窗口 | User32.lib |
| `GetMessage/TranslateMessage/DispatchMessage` | 消息循环 | User32.lib |
| `DefWindowProc` | 默认窗口过程 | User32.lib |
| `SendMessage/PostMessage` | 同步/异步发送消息 | User32.lib |
| `SetWindowLongPtr/GetWindowLongPtr` | 保存窗口关联数据 | User32.lib |
| `CreateDialog/DialogBox` | 对话框 | User32.lib |
| `InitCommonControlsEx` | 初始化通用控件 | Comctl32.lib |
| `DwmSetWindowAttribute` | DWM 属性 | Dwmapi.lib |
| `SetProcessDpiAwarenessContext` | DPI 感知 | User32.lib |

## 使用要点

- 查看每个 API 文档中的 **Header / Library / DLL / Minimum supported client**。
- C++ 默认建议走 Unicode 版本，避免 ANSI 代码页问题。
- 明确资源释放函数：`CloseHandle`、`LocalFree`、`CoTaskMemFree`、`DeleteObject`、`Release` 等不可混用。
- 失败处理：Win32 多数用 `GetLastError()`，COM 多数用 `HRESULT`。
