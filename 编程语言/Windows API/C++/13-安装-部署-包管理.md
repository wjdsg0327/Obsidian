# C++：安装、部署、包管理

MSI、Restart Manager、Side-by-side、AppX/MSIX、开发者许可。

## 常用 API 速查

| API | 用途 | 库/模块 |
|---|---|---|
| `MsiInstallProduct` | 安装 MSI | Msi.lib |
| `MsiOpenProduct/MsiGetProductInfo` | MSI 产品信息 | Msi.lib |
| `MsiConfigureProduct` | 配置/卸载 MSI | Msi.lib |
| `CreateActCtx/ActivateActCtx` | SxS 清单 | Kernel32.lib |
| `RmStartSession/RmRegisterResources` | Restart Manager | Rstrtmgr.lib |
| `Package APIs` | AppX/MSIX 查询 | Kernel32/Appmodel |

## 使用要点

- 查看每个 API 文档中的 **Header / Library / DLL / Minimum supported client**。
- C++ 默认建议走 Unicode 版本，避免 ANSI 代码页问题。
- 明确资源释放函数：`CloseHandle`、`LocalFree`、`CoTaskMemFree`、`DeleteObject`、`Release` 等不可混用。
- 失败处理：Win32 多数用 `GetLastError()`，COM 多数用 `HRESULT`。
