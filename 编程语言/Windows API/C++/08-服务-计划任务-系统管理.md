# C++：服务、计划任务、系统管理

Windows Service、SCM、事件日志、WMI、组策略、性能计数器。

## 常用 API 速查

| API | 用途 | 库/模块 |
|---|---|---|
| `OpenSCManager` | 打开服务控制管理器 | Advapi32.lib |
| `CreateService/OpenService` | 创建/打开服务 | Advapi32.lib |
| `StartService/ControlService` | 启动/控制服务 | Advapi32.lib |
| `QueryServiceStatusEx` | 查询服务状态 | Advapi32.lib |
| `RegisterServiceCtrlHandlerEx` | 服务控制回调 | Advapi32.lib |
| `StartServiceCtrlDispatcher` | 服务入口调度 | Advapi32.lib |
| `OpenEventLog/ReportEvent` | 旧事件日志 | Advapi32.lib |
| `EvtQuery/EvtNext` | Windows Event Log | Wevtapi.lib |
| `PdhOpenQuery/PdhAddCounter` | 性能计数器 | Pdh.lib |
| `CoCreateInstance(IWbemLocator)` | WMI COM 入口 | Wbemuuid.lib |

## 使用要点

- 查看每个 API 文档中的 **Header / Library / DLL / Minimum supported client**。
- C++ 默认建议走 Unicode 版本，避免 ANSI 代码页问题。
- 明确资源释放函数：`CloseHandle`、`LocalFree`、`CoTaskMemFree`、`DeleteObject`、`Release` 等不可混用。
- 失败处理：Win32 多数用 `GetLastError()`，COM 多数用 `HRESULT`。
