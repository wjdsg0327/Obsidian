# 06-动画和资源管理

## 资源是什么

Cocos 里的资源包括：

- 图片 texture/spriteFrame
- 音频 audio
- 字体 font
- 场景 scene
- 预制体 prefab
- 动画 animation
- 脚本 script

资源管理的目标是：不要乱，能复用。

## Sprite 显示图片

一个图片显示出来，通常是：

```text
Node
└─ Sprite 组件
   └─ SpriteFrame 图片资源
```

你把图片拖进项目后，选中节点的 Sprite 组件，把 SpriteFrame 设成这张图。

## 预制体 Prefab

预制体适合反复生成的对象：

- 子弹
- 怪物
- 金币
- 特效

创建方式：

1. 在场景里做好一个 Coin 节点
2. 从 Hierarchy 拖到 Assets/prefabs
3. 得到 Coin.prefab

代码生成：

```ts
import { _decorator, Component, Prefab, instantiate, Node } from 'cc';
const { ccclass, property } = _decorator;

@ccclass('Spawner')
export class Spawner extends Component {
  @property(Prefab)
  coinPrefab: Prefab | null = null;

  @property(Node)
  container: Node | null = null;

  spawn() {
    if (!this.coinPrefab || !this.container) return;
    const coin = instantiate(this.coinPrefab);
    this.container.addChild(coin);
    coin.setPosition(0, 300, 0);
  }
}
```

## 简单动画

Cocos 可以用 Animation 编辑器做关键帧动画。

适合：

- 按钮缩放
- 金币旋转
- 角色待机
- UI 弹出

基本流程：

1. 选中节点
2. 添加 Animation 组件
3. 创建 Animation Clip
4. 打关键帧
5. 设置循环或播放方式

## 用代码做简单动画

很多轻量动画可以直接用 tween：

```ts
import { _decorator, Component, tween, Vec3 } from 'cc';
const { ccclass } = _decorator;

@ccclass('PopAnim')
export class PopAnim extends Component {
  start() {
    this.node.setScale(new Vec3(0, 0, 1));
    tween(this.node)
      .to(0.2, { scale: new Vec3(1.2, 1.2, 1) })
      .to(0.1, { scale: new Vec3(1, 1, 1) })
      .start();
  }
}
```

## 资源引用方式

入门优先用 Inspector 拖引用：

```ts
@property(Prefab)
coinPrefab: Prefab | null = null;
```

这样清晰，不容易写错路径。

后面再学动态加载资源。

## 常见错误

### 删除资源导致引用丢失

Inspector 里出现 Missing，说明原来拖的资源没了或移动后引用出问题。

### Prefab 改了场景对象没同步

要理解 Prefab 和场景实例的关系。初学时，如果乱了，重新拖一个 Prefab 实例最省事。

### 资源目录混乱

素材一多就找不到。早点按类型分文件夹。
