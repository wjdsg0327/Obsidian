# 04-节点控制：移动、旋转、缩放和输入

## 控制节点就是控制 Transform

节点最常改的三个东西：

- 位置 position
- 旋转 rotation
- 缩放 scale

在 2D 游戏里，最常用的是位置。

## 设置位置

```ts
this.node.setPosition(100, 0, 0);
```

读取位置：

```ts
const pos = this.node.position;
console.log(pos.x, pos.y);
```

修改位置时建议 clone：

```ts
const pos = this.node.position.clone();
pos.x += 10;
this.node.setPosition(pos);
```

## 每帧移动

```ts
update(deltaTime: number) {
  const pos = this.node.position.clone();
  pos.x += 200 * deltaTime;
  this.node.setPosition(pos);
}
```

这表示每秒向右移动 200 像素左右。

## 键盘输入

Web/PC 预览可以用键盘：

```ts
import { _decorator, Component, input, Input, EventKeyboard, KeyCode } from 'cc';
const { ccclass } = _decorator;

@ccclass('PlayerController')
export class PlayerController extends Component {
  private dir = 0;
  speed = 300;

  onLoad() {
    input.on(Input.EventType.KEY_DOWN, this.onKeyDown, this);
    input.on(Input.EventType.KEY_UP, this.onKeyUp, this);
  }

  onDestroy() {
    input.off(Input.EventType.KEY_DOWN, this.onKeyDown, this);
    input.off(Input.EventType.KEY_UP, this.onKeyUp, this);
  }

  onKeyDown(event: EventKeyboard) {
    if (event.keyCode === KeyCode.ARROW_LEFT) this.dir = -1;
    if (event.keyCode === KeyCode.ARROW_RIGHT) this.dir = 1;
  }

  onKeyUp(event: EventKeyboard) {
    if (event.keyCode === KeyCode.ARROW_LEFT && this.dir === -1) this.dir = 0;
    if (event.keyCode === KeyCode.ARROW_RIGHT && this.dir === 1) this.dir = 0;
  }

  update(dt: number) {
    const pos = this.node.position.clone();
    pos.x += this.dir * this.speed * dt;
    this.node.setPosition(pos);
  }
}
```

## 触摸/鼠标输入

小游戏更常用触摸。简单做法：点哪里，玩家移动到哪里：

```ts
import { _decorator, Component, input, Input, EventTouch, Vec3 } from 'cc';
const { ccclass } = _decorator;

@ccclass('TouchMove')
export class TouchMove extends Component {
  onLoad() {
    input.on(Input.EventType.TOUCH_START, this.onTouch, this);
  }

  onDestroy() {
    input.off(Input.EventType.TOUCH_START, this.onTouch, this);
  }

  onTouch(event: EventTouch) {
    const ui = event.getUILocation();
    this.node.setWorldPosition(new Vec3(ui.x, ui.y, 0));
  }
}
```

实际项目里坐标转换会更细，但入门先理解“监听输入 -> 改节点位置”。

## 限制移动范围

玩家不能跑出屏幕：

```ts
const minX = -320;
const maxX = 320;
pos.x = Math.max(minX, Math.min(maxX, pos.x));
```

## 常见错误

### 忘记解绑事件

`onLoad` 里 `input.on`，最好在 `onDestroy` 里 `input.off`。

### 移动太快或太慢

检查是否乘了 `deltaTime`。

### 坐标系混乱

`setPosition` 是本地坐标，`setWorldPosition` 是世界坐标。初学尽量让移动节点都在 Canvas 下，减少混乱。
