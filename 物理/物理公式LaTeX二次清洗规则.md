# 物理公式 LaTeX 二次清洗规则

> 整理时间：2026-06-08  
> 用途：配合 MinerU / marker 抽取后的 Markdown，对物理公式进行二次清洗。  
> 原则：**PDF 转换结果只当初稿，公式需要规则化重写。**

---

## 1. 清洗目标

把 OCR / PDF 转换工具输出的普通文本公式、乱码公式、错误上下标，统一整理成可读、可检索、适合 Obsidian / Markdown / RAG 使用的 LaTeX。

---

## 2. 基础格式

- 行内公式：`$...$`
- 独立公式：

```markdown
$$
...
$$
```

- 不确定公式：`【公式待核对】`
- 不确定数值或单位：`【数值待核对】`、`【单位待核对】`

## 4. OCR 常见错误修正

| OCR 输出 | 清洗后 |
|---|---|
| `△v` | `\Delta v` |
| `Δt` | `\Delta t` |
| `I2Rt` | `I^2Rt` |
| `10 的 3 次方` | `10^3` |
| `x10^3` | `\times 10^3` |
| `kg/m3` | `\mathrm{kg/m^3}` |
| `N/m2` | `\mathrm{N/m^2}` |
| `m/s` | `\mathrm{m/s}` 或 `\mathrm{m\,s^{-1}}` |
| `ms-1` | `\mathrm{m\,s^{-1}}` |
| `kgm-3` | `\mathrm{kg\,m^{-3}}` |
| `W有/W总` | `\frac{W_{\text{有}}}{W_{\text{总}}}` |

## 5. 清洗策略

### 5.1 可以直接规则替换

适合稳定、明确的公式：

- `F=ma`
- `P=UI`
- `Q=I2Rt`
- `rho=m/V`
- `p=F/S`

### 5.2 需要结合上下文判断

这些符号可能有多种含义：

- `p`：可能是压强，也可能是动量。
- `P`：可能是功率，也可能是概率/压力符号。
- `Q`：可能是热量，也可能是电荷量。
- `W`：可能是功，也可能是能量单位瓦特误识别。
- `phi`：可能是 `\phi`，也可能是磁通量 `\Phi`。

遇到不确定情况，不要硬改，标记：`【公式待核对】`。

## 6. 推荐脚本规则结构

```python
FORMULA_RULES = {
    r"v\s*=\s*s\s*/\s*t": r"$v = \\frac{s}{t}$",
    r"a\s*=\s*[△Δ]v\s*/\s*[△Δ]t": r"$a = \\frac{\\Delta v}{\\Delta t}$",
    r"F\s*=\s*ma": r"$F = ma$",
    r"p\s*=\s*F\s*/\s*S": r"$p = \\frac{F}{S}$",
    r"rho\s*=\s*m\s*/\s*V": r"$\\rho = \\frac{m}{V}$",
    r"I\s*=\s*U\s*/\s*R": r"$I = \\frac{U}{R}$",
    r"Q\s*=\s*I\s*2\s*R\s*t": r"$Q = I^2Rt$",
}
```

实际脚本里建议：

1. 先做字符标准化。
2. 再做单位修正。
3. 再做公式规则替换。
4. 最后输出 `【公式待核对】` 列表。
