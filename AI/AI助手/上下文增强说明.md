# 上下文增强说明

聊天窗口本身有长度限制，但可以通过“外置记忆 + 按需读取”提高有效上下文。

## 推荐用法

1. 重要偏好、固定路径、长期约定：写入 `丽华长期记忆.md` 或 OpenClaw 的 `MEMORY.md` / `TOOLS.md`。
2. 长项目：建立项目专属说明文件，例如：
   - `项目说明.md`
   - `当前状态.md`
   - `TODO.md`
   - `决策记录.md`
3. 每次继续项目时，让我先读取对应项目状态文件。
4. 大量资料不要全部塞进聊天，放进知识库后让我检索和精读。
