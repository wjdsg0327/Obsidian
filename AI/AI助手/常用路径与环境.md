# 常用路径与环境

## 知识库

- Windows：`D:\老王`
- WSL：`/mnt/d/老王/`

## Docker 数据

- Windows：`D:\work\docker`
- WSL：`/mnt/d/work/docker`
- 约定：每个服务/项目单独建子目录保存持久化数据。

## OpenClaw 工作区

- `/root/.openclaw/workspace`
- 长期个人偏好：`MEMORY.md`
- 环境工具说明：`TOOLS.md`
- 每日原始记录：`memory/YYYY-MM-DD.md`
