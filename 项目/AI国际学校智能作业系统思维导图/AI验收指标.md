# AI验收指标

> 本文档定义 AI 国际学校智能作业系统 MVP 中 OCR、智能组题、AI 批改和学情总结的验收指标。

## 1. 总体原则

- AI 不以“看起来能用”为验收标准，必须有样本集和量化指标。
- MVP 中 AI 批改是教师辅助工具，低置信度必须进入人工复核。
- AI 验收样本应覆盖清晰图片、模糊图片、不同书写、不同题型、不同难度。

## 2. OCR 验收指标

| 指标 | MVP 目标 | 说明 |
|---|---:|---|
| 清晰图片文字识别可读率 | ≥ 95% | 教师/学生正常拍照上传 |
| 题号识别准确率 | ≥ 90% | 能正确分辨题号和答案块 |
| 模糊图片低置信度识别率 | ≥ 90% | 模糊时应提示人工/重传，而不是乱识别 |
| PDF 页序保持准确率 | ≥ 99% | 多页提交顺序不乱 |

## 3. 客观题批改指标

| 指标 | MVP 目标 |
|---|---:|
| 选择题批改准确率 | ≥ 99% |
| 判断题批改准确率 | ≥ 99% |
| 填空题标准答案匹配准确率 | ≥ 95% |
| 答案为空识别准确率 | ≥ 98% |

## 4. 计算题 / 简答题指标

| 指标 | MVP 目标 | 说明 |
|---|---:|---|
| 简单计算题最终答案判断准确率 | ≥ 90% | 含单位和有效数字 |
| 简单计算题评分一致率 | ≥ 85% | 与教师评分相比 |
| 简答题评分一致率 | ≥ 80% | MVP 仅辅助，不直接最终 |
| 低置信度召回率 | ≥ 90% | 不确定时进入人工复核 |

## 5. 智能组题指标

| 指标 | MVP 目标 |
|---|---:|
| 硬条件匹配准确率 | 100% |
| 知识点匹配准确率 | ≥ 95% |
| 难度分布误差 | ≤ 10% |
| 近期重复题过滤准确率 | ≥ 95% |
| 教师接受率 | ≥ 70% |

## 6. 学情总结指标

| 指标 | MVP 目标 |
|---|---:|
| 不编造数据 | 100% |
| 薄弱知识点引用准确率 | ≥ 95% |
| 建议与数据相关性 | ≥ 85% 教师认可 |
| 生成耗时 | ≤ 10 秒 |

## 7. 验收样本建议

MVP 验收至少准备：

- 100 份学生提交样本。
- 1000 道客观题答案。
- 200 道填空题答案。
- 100 道简单计算题。
- 50 道简答题。
- 20 份模糊/异常上传样本。

## 8. 不通过处理

如果某项指标不达标：

1. 不阻断整个平台上线，但相关 AI 能力必须降级。
2. 降级方式包括：人工复核、关闭自动最终分、只展示建议。
3. 记录问题样本，用于后续优化 Prompt、模型或规则引擎。
