# 题库标签体系

## 1. 标签设计目标

题库标签要同时服务于：

- 教师筛题。
- AI 智能组题。
- 自动批改。
- 学情分析。
- 错题归因。
- 题库版权追溯。

## 2. 标签分层

### 2.1 基础标签

| 标签 | 示例 | 是否必填 |
|---|---|---:|
| 课程体系 | A-Level / IGCSE / AP / DSE / IB | 是 |
| 年级/阶段 | AS / A2 / Grade 10 | 否 |
| 学科 | Chemistry / Physics / Math | 是 |
| 考局 | CIE / Edexcel / AQA / CB / IBO | 否 |
| 教材版本 | Cambridge A Level Chemistry | 否 |
| 章节 | Atomic Structure | 否 |

### 2.2 知识点标签

建议三级结构：

```text
一级：Physical Chemistry
  二级：Atomic Structure
    三级：Isotopes
    三级：Mass Spectrometry
  二级：Chemical Bonding
一级：Organic Chemistry
一级：Inorganic Chemistry
```

字段建议：

- `knowledge_level1`
- `knowledge_level2`
- `knowledge_level3`
- `knowledge_point_id`

### 2.3 题型标签

- 选择题
- 判断题
- 填空题
- 计算题
- 简答题
- 论述题
- 画图题
- 实验题

### 2.4 难度标签

| 难度 | 定义 | 推荐用途 |
|---:|---|---|
| 1 | 概念识记 | 初次上课、基础巩固 |
| 2 | 基础应用 | 初次上课、课后练习 |
| 3 | 中等综合 | 复习课、阶段检测 |
| 4 | 较难综合 | 强化训练 |
| 5 | 高难压轴 | 冲刺、竞赛、拓展 |

难度来源优先级：

1. 考局 examiner report / 官方数据。
2. 历史学生正确率。
3. 教师人工标注。
4. AI 初始估计。

### 2.5 来源标签

| 标签 | 示例 |
|---|---|
| source_type | 教材 / 教辅 / 真题 / 官方模拟题 / 自定义 |
| source_name | CIE 2022 May/June Paper 12 |
| exam_year | 2022 |
| exam_season | May/June |
| paper_code | 9701/12 |
| question_no | Q3 |
| copyright_source | Cambridge official paper |

### 2.6 频率标签

- 高频：近 5 年出现频率高。
- 中频：近 5 年出现频率中等。
- 低频：偶尔出现。

建议字段：`exam_frequency_level`。

### 2.7 AI 辅助标签

- `ai_tag_confidence`：AI 打标置信度。
- `ai_tag_model`：打标模型。
- `review_status`：审核状态。
- `reviewer_id`：审核人。

## 3. 标签审核规则

- AI 自动打标后状态为 `pending_review`。
- 知识点、题型、难度、版权来源必须人工确认。
- 置信度低于 0.85 的标签重点标红。
- 审核通过后才能进入公共题库。

## 4. 标签在业务中的使用

### 4.1 智能组题

使用标签：课程体系、学科、考局、知识点、题型、难度、来源、真题年份。

### 4.2 学情分析

使用标签：知识点、题型、难度。

### 4.3 错题集

使用标签：学生、知识点、题型、错误次数、掌握状态。

### 4.4 真题隔离

使用标签：source_type、exam_year、paper_code。

## 5. MVP 建议

第一版至少实现：

- 课程体系
- 学科
- 考局
- 三级知识点
- 题型
- 难度
- 来源类型
- 版权来源
- 审核状态

其他标签可后续增强。
