# 接口设计草案

## 1. 约定

- Base URL：`/api/v1`
- 数据格式：JSON
- 鉴权方式：Bearer Token
- 时间格式：ISO 8601
- 分页参数：`page`、`pageSize`

## 2. 通用响应

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

错误示例：

```json
{
  "code": 40001,
  "message": "无权限访问该资源",
  "data": null
}
```

## 3. 账号与权限

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | /auth/login | 登录 |
| POST | /auth/logout | 退出 |
| GET | /auth/me | 当前用户信息 |
| POST | /users/import-teachers | 批量导入教师 |
| POST | /users/import-students | 批量导入学生 |
| GET | /users | 用户列表 |
| PATCH | /users/{id}/status | 启用/禁用账号 |
| POST | /users/{id}/reset-password | 重置密码 |
