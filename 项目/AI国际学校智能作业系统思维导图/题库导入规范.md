# 题库导入规范

> 用于规范 Excel / Word / PDF 题库导入，保证题目可检索、可组题、可批改、可追溯版权来源。

## 1. 导入方式

| 格式 | MVP 是否支持 | 用途 |
|---|---:|---|
| Excel | 优先支持 | 结构化批量导入题目和标签 |
| Word | 支持 | 教师自定义题目、历史资料导入 |
| PDF | 支持 | 真题、扫描资料、教材教辅资料导入 |
| 图片 | 后置 | 单题拍照导入 |

## 3. 字段枚举建议

### 3.1 课程体系

- A-Level
- IGCSE
- AP
- DSE
- IB

### 3.2 题型

- 选择题
- 判断题
- 填空题
- 计算题
- 简答题
- 论述题
- 画图题
- 实验题

### 3.3 难度

| 难度 | 含义 |
|---:|---|
| 1 | 基础概念，适合初学 |
| 2 | 基础应用，常规练习 |
| 3 | 中等综合，需要多步思考 |
| 4 | 较难，综合应用或易错点 |
| 5 | 高难，压轴/竞赛/综合拓展 |

### 3.4 来源类型

- 教材
- 教辅
- 真题
- 官方模拟题
- 教师自定义
- AI 生成后人工审核

## 4. Word / PDF 导入规则

### 4.1 推荐排版

```text
[题目编号]
[题干]
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：B
解析：……
知识点：Atomic Structure / Isotopes
难度：2
来源：2022 CIE 9701/12 Q3
```

### 4.2 AI 自动识别流程

1. 上传文件。
2. 根据文件类型选择解析方式：
   - Excel：按模板字段直接读取。
   - Word：提取段落、表格和图片后进入题目块识别。
   - PDF：优先进入文档解析 Provider；扫描版 PDF 同时触发 OCR。
3. OCR/文档解析提取题目块。
4. AI 识别题干、选项、答案、解析、来源。
5. AI 自动打标签。
6. 进入人工审核。
7. 审核通过后进入正式题库。

### 4.3 PDF 文档解析 Provider 候选：MinerU

MinerU 可作为 PDF/扫描资料解析的 PoC 候选方案，用于将真题、教材教辅、扫描题库等资料预处理为 Markdown / JSON / 图片资源，再交给 AI 做题目块切分和字段抽取。

建议使用范围：

- 真题 PDF、教材教辅 PDF、扫描资料导入。
- 多栏排版、表格、公式、图片较多的题库资料。
- 需要保留页码、题号、公式、图表资源的导入场景。

输出进入题库前，必须经过人工审核，不能直接入正式题库。

建议中间结果字段：

| 字段 | 说明 |
|---|---|
| source_file_id | 原始文件 ID |
| provider | 解析供应商，例如 mineru |
| page_no | 页码 |
| markdown | 页面或题目块 Markdown 内容 |
| block_type | text / table / formula / image |
| bbox | 版面坐标，可选 |
| confidence | 解析置信度，可选 |
| asset_paths | 图片、公式截图等资源路径 |

MVP 建议先做 PoC：选择 3-5 份代表性 PDF（真题、教材页、扫描件），验证解析准确率、耗时、部署资源和人工修正成本。

## 5. 校验规则

导入时必须校验：

- `course_system`、`subject`、`question_type`、`difficulty`、`stem`、`answer`、`score`、`copyright_source` 不得为空。
- `difficulty` 必须为 1-5。
- 选择题必须有 `options`。
- 真题必须尽量填写 `exam_board`、`exam_year`、`paper_code`。
- 同一来源下 `paper_code + question_no` 不建议重复。
- 题目内容重复度过高时提示疑似重复。

## 6. 审核流程

| 状态 | 说明 |
|---|---|
| draft | 导入草稿 |
| ai_tagged | AI 已识别并打标 |
| pending_review | 待人工审核 |
| approved | 审核通过，可用于组题 |
| rejected | 审核拒绝 |
| archived | 已归档，不再用于新作业 |

## 7. 版权管理

- 每道题必须记录 `copyright_source`。
- 官方真题需记录考局、年份、试卷编号。
- 教材/教辅题需记录教材名、版本、章节、页码。
- 未确认版权的题目只能进入个人题库或待审核状态，不进入公共题库。
- 导出题库时应保留版权来源字段。

## 8. 导入失败反馈

导入失败文件应返回错误报告，至少包含：

| 行号 | 字段 | 错误原因 | 建议修复 |
|---|---|---|---|
| 12 | difficulty | 难度为空 | 填写 1-5 |
| 18 | options | 选择题缺少选项 | 按 A/B/C/D 格式补充 |
