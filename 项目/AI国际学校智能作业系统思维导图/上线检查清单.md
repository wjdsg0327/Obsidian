# 上线检查清单

## 1. 功能检查

- [ ] 登录、退出、Token 过期处理正常。
- [ ] 超级管理员、学校管理员、教师、学生、家长菜单正确。
- [ ] 教师/学生批量导入可用。
- [ ] 班级创建、编辑、归档可用。
- [ ] 题库导入、审核、筛选可用。
- [ ] 教师可创建草稿、发布作业。
- [ ] 学生可查看并提交作业。
- [ ] AI 批改可触发并生成记录。
- [ ] 教师可复核、改分、写评语。
- [ ] 学生可查看最终批改反馈。
- [ ] 错题自动生成。
- [ ] 班级/学生学情页面可用。
- [ ] 操作日志可查询。

## 2. 权限检查

- [ ] 学校管理员只能访问本校数据。
- [ ] 教师只能访问所教班级数据。
- [ ] 学生只能访问本人作业和学情。
- [ ] 家长只能访问绑定子女数据。
- [ ] 未登录不能访问业务接口。
- [ ] 数据导出有权限校验和日志。

## 3. 数据检查

- [ ] 数据库迁移脚本已执行。
- [ ] 初始管理员账号已创建。
- [ ] 基础课程体系枚举已初始化。
- [ ] 知识点树已初始化。
- [ ] 测试题库已导入并审核。
- [ ] 测试学校、教师、学生、班级数据已准备。

## 4. AI 检查

- [ ] OCR 服务可用。
- [ ] AI 批改服务可用。
- [ ] 智能组题服务可用。
- [ ] AI 调用失败有兜底提示。
- [ ] AI 调用日志记录完整。
- [ ] 教师限额生效。
- [ ] 成本统计可查看。

## 5. 性能检查

- [ ] 首页加载 ≤ 2 秒。
- [ ] 常规接口响应 ≤ 1 秒。
- [ ] 数据库查询无明显慢查询。
- [ ] 100MB 文件上传策略验证通过。
- [ ] 批量 AI 批改异步执行，不阻塞页面。

## 6. 安全检查

- [ ] HTTPS 已启用。
- [ ] 密码非明文存储。
- [ ] API 防 SQL 注入。
- [ ] 前端防 XSS。
- [ ] CSRF 防护或 Token 策略确认。
- [ ] 文件上传类型和大小限制。
- [ ] 敏感配置不在前端代码。
- [ ] 备份文件权限正确。

## 7. 运维检查

- [ ] 生产配置已确认。
- [ ] 数据库备份策略已启用。
- [ ] 日志路径和保留策略已确认。
- [ ] 监控告警已配置。
- [ ] 回滚方案已演练。
- [ ] 上线窗口已确认。
- [ ] 负责人联系方式已记录。

## 8. 交付资料检查

- [ ] 需求文档。
- [ ] 数据库设计说明。
- [ ] 接口文档。
- [ ] 部署运维方案。
- [ ] 测试报告。
- [ ] 管理员操作说明。
- [ ] 教师操作说明。
- [ ] 学生操作说明。


## 9. 回滚与恢复演练

- [ ] 上线前已备份生产数据库。
- [ ] 数据库迁移脚本已在预发布环境验证。
- [ ] 数据库迁移具备回滚方案或恢复点。
- [ ] 上一版本应用镜像/包可立即恢复。
- [ ] 文件存储备份和恢复流程已验证。
- [ ] 至少完成一次测试环境备份恢复演练。
- [ ] AI 服务不可用时，系统可切换为人工批改流程。
- [ ] 回滚负责人、操作人、确认人已明确。

## 10. AI 验收检查

- [ ] OCR 清晰图片可读率达到验收目标。
- [ ] 客观题批改准确率达到验收目标。
- [ ] 低置信度样本能进入人工复核。
- [ ] AI 调用日志包含模型、耗时、成本、状态。
- [ ] 教师限额和超额提醒验证通过。
