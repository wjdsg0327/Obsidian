# AI批改规则与 Prompt 草案

## 1. 批改总原则

- AI 批改只作为辅助，最终成绩以教师确认结果为准。
- 客观题优先使用规则引擎精确批改，AI 用于 OCR 纠错和解释生成。
- 主观题必须输出置信度，低置信度进入人工复核。
- 复杂论述、画图、实验题 MVP 阶段默认人工批改。
- 所有 AI 输出必须可追溯：模型、时间、输入摘要、置信度、成本。

## 2. 题型批改规则

| 题型 | 批改方式 | 人工复核条件 |
|---|---|---|
| 选择题 | 标准答案精确匹配 | OCR 识别不清、答案多选异常 |
| 判断题 | 标准答案精确匹配 | OCR 识别不清 |
| 填空题 | 标准化后匹配，允许等价表达 | 同义词/格式不确定 |
| 计算题 | 最终答案 + 关键步骤评分 | 步骤缺失、单位不明确、置信度低 |
| 简答题 | 关键词 + LLM 评分 | 置信度低于 0.85 |
| 论述题 | 暂不自动给最终分 | 始终人工复核 |
| 画图题 | 暂不自动给最终分 | 始终人工复核 |
| 实验题 | 暂不自动给最终分 | 始终人工复核 |

## 3. OCR Prompt 草案

```text
你是国际课程作业 OCR 助手。请从学生上传的作业图片中识别答案内容。
要求：
1. 按页码和题号输出。
2. 不要补全看不清的内容；看不清写 [unclear]。
3. 保留数学公式、化学方程式、单位。
4. 输出 JSON，不要输出解释文字。

JSON 格式：
{
  "pages": [
    {
      "pageNo": 1,
      "answers": [
        {
          "questionNo": "1",
          "text": "识别出的学生答案",
          "confidence": 0.0
        }
      ]
    }
  ]
}
```

## 4. 计算题批改 Prompt 草案

```text
你是 A-Level/IGCSE 化学教师，请根据标准答案批改学生答案。

输入包括：
- 题目
- 标准答案
- 评分规则
- 学生答案 OCR 文本

要求：
1. 严格按照评分规则给分。
2. 如果最终答案正确但步骤缺失，按评分规则扣分。
3. 注意单位、有效数字、化学符号、方程式配平。
4. 若 OCR 不清或无法判断，needsManualReview=true。
5. 输出 JSON，不要输出 Markdown。

输出格式：
{
  "score": 0,
  "maxScore": 0,
  "isCorrect": false,
  "comment": "给学生看的简短反馈",
  "reason": "给教师看的评分依据",
  "confidence": 0.0,
  "needsManualReview": false
}
```

## 5. 简答题批改 Prompt 草案

```text
你是国际课程作业批改助手。请根据评分点批改学生简答题。

评分要求：
- 只根据评分点给分，不要因为表达华丽额外加分。
- 学生表达同义但科学含义正确，可以给分。
- 关键概念错误必须扣分。
- 无法确定时标记人工复核。

输出 JSON：
{
  "matchedPoints": ["评分点1"],
  "missedPoints": ["评分点2"],
  "score": 0,
  "maxScore": 0,
  "feedback": "反馈",
  "confidence": 0.0,
  "needsManualReview": false
}
```

## 6. 学情总结 Prompt 草案

```text
你是国际学校学情分析助手。请根据班级作业数据生成教师可用的学情总结。

要求：
1. 先总结整体表现。
2. 列出优势知识点和薄弱知识点。
3. 给出下一节课教学建议。
4. 避免夸大，不要编造数据。
5. 输出 JSON。

输出：
{
  "summary": "整体表现",
  "strengths": [],
  "weaknesses": [],
  "teachingSuggestions": [],
  "recommendedPractice": []
}
```

## 7. 人工复核触发规则

- `confidence < 0.85`
- OCR 文本包含 `[unclear]`
- 题型为论述题、画图题、实验题
- AI 分数与规则引擎分数差异超过 20%
- 学生答案为空但上传文件非空
- 模型调用失败或超时

## 8. 输出入库规则

- AI 原始输出保存到日志或结果扩展字段。
- `grading_records.ai_score` 保存 AI 分数。
- `grading_records.final_score` 保存教师确认分数。
- `needs_manual_review=true` 时，学生端显示“批改中”。
