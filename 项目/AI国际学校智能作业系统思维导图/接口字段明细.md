# 接口字段明细

> Base URL：`/api/v1`  
> 鉴权：`Authorization: Bearer <token>`

## 1. 登录

### POST /auth/login

请求：

```json
{
  "username": "teacher001",
  "password": "123456"
}
```

响应：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "jwt-token",
    "user": {
      "id": 1,
      "schoolId": 10,
      "username": "teacher001",
      "realName": "王老师",
      "role": "teacher"
    }
  }
}
```

## 2. 班级

### GET /classes

查询参数：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| schoolId | number | 否 | 管理员可传 |
| grade | string | 否 | 年级 |
| subject | string | 否 | 学科 |
| status | string | 否 | active/archived |

响应字段：

```json
{
  "items": [
    {
      "id": 1,
      "name": "AS 化学 1 班",
      "grade": "AS",
      "subject": "Chemistry",
      "courseSystem": "A-Level",
      "examBoard": "CIE",
      "studentCount": 28,
      "headTeacherName": "王老师",
      "status": "active"
    }
  ],
  "total": 1
}
```

### POST /classes

```json
{
  "name": "AS 化学 1 班",
  "grade": "AS",
  "subject": "Chemistry",
  "courseSystem": "A-Level",
  "examBoard": "CIE",
  "textbookVersion": "Cambridge International AS & A Level Chemistry",

```

## 3. 题库

### GET /questions

查询参数：

| 字段 | 类型 | 说明 |
|---|---|---|
| courseSystem | string | 课程体系 |
| subject | string | 学科 |
| examBoard | string | 考局 |
| knowledgePointId | number | 知识点 |
| questionType | string | 题型 |
| difficulty | number | 难度 1-5 |
| reviewStatus | string | 审核状态 |

响应：

```json
{
  "items": [
    {
      "id": 1001,
      "stem": "Which particle has the smallest mass?",
      "options": ["proton", "neutron", "electron", "alpha particle"],
      "answer": "C",
      "explanation": "Electron has the smallest mass.",
      "difficulty": 2,
      "questionType": "选择题",
      "knowledgePointName": "Atomic Structure",
      "sourceType": "真题",
      "copyrightSource": "CIE official paper"
    }
  ],
  "total": 1
}
```

### POST /questions/import

请求：`multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| file | file | 是 | Excel/Word/PDF |
| sourceType | string | 是 | 教材/教辅/真题/自定义 |
| autoTag | boolean | 否 | 是否 AI 自动打标 |

响应：

```json
{
  "importId": "IMP20260607001",
  "successCount": 120,
  "failedCount": 3,
  "errorReportUrl": "/files/import-errors.xlsx"
}
```

```json

  "courseSystem": "A-Level",
```

```json
      "score": 0.91,
      "reason": "匹配 Atomic Structure，难度 2 星，适合初次上课巩固"
```

## 9. 错误码建议

| code | 说明 |
|---:|---|
| 0 | 成功 |
| 40000 | 请求参数错误 |
| 40100 | 未登录或 Token 失效 |
| 40300 | 无权限 |
| 40400 | 资源不存在 |
| 40900 | 数据冲突 |
| 50000 | 系统错误 |
| 60000 | AI 服务异常 |
| 60001 | AI 限额不足 |


## 10. 文件上传接口

### POST /files/upload

请求：`multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| file | file | 是 | 上传文件 |
| bizType | string | 是 | submission/question_import/pdf/courseware/audio |
| bizId | number | 否 | 业务 ID |

响应：

```json
{
  "fileId": 101,
  "fileUrl": "/files/101",
  "originalName": "homework.jpg",
  "mimeType": "image/jpeg",
  "sizeBytes": 204800,
  "status": "uploaded"
}
```

## 11. 作业发布范围接口

### POST /assignments/{id}/targets

```json
{
  "targets": [
    {"targetType": "student", "targetId": 1001},
    {"targetType": "student", "targetId": 1002}
  ]
}
```

说明：

- `targetType=class`：全班。
- `targetType=student`：指定学生。
- `targetType=group`：指定分组，MVP 可预留。

```json
      "id": 1,
```

标记通知已读。

## 13. AI 成本查询接口

### GET /admin/ai-costs

查询参数：

| 字段 | 类型 | 说明 |
|---|---|---|
| schoolId | number | 学校 ID |
| teacherId | number | 教师 ID |
| taskType | string | ocr/grade/recommend/summary |
| startDate | string | 开始日期 |
| endDate | string | 结束日期 |

响应：

```json
{
  "totalCost": 82.8,
  "totalCalls": 3600,
  "byTaskType": [
    {"taskType": "ocr", "calls": 1000, "cost": 20.5},
    {"taskType": "grade", "calls": 2500, "cost": 55.3}
  ]
}
```

## 14. 数据权限要求

- 所有接口均以后端当前用户身份计算数据范围。
- 前端传入的 `schoolId/classId/studentId` 只能作为筛选条件，不能作为权限依据。
- 越权访问统一返回 `40300`。
