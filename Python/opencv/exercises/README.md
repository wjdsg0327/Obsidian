# OpenCV 配套练习

这个目录提供与教程章节一一对应的练习题，建议在学完一章后立刻完成当章练习。

做练习前请先安装依赖：

```bash
pip install -r requirements.txt
```

## 建议顺序

1. 先完成基础练习，熟悉 API。
2. 再做进阶练习，把多个知识点串起来。
3. 最后回答思考题，把“会写”升级成“会解释”。

## 文件索引

- `ch00_exercises.md`: 环境准备与第一段 OpenCV 代码
- `ch01_exercises.md`: 图像读写、像素访问与 ROI
- `ch02_exercises.md`: 绘图、文字与颜色空间
- `ch03_exercises.md`: 缩放、旋转与几何变换
- `ch04_exercises.md`: 滤波、阈值与图像增强
- `ch05_exercises.md`: 边缘、形态学与轮廓分析
- `ch06_exercises.md`: 摄像头、视频与实时处理
- `ch07_exercises.md`: 特征提取、匹配、霍夫变换与模板匹配
- `ch08_exercises.md`: 图像分割与高级处理
- `ch09_exercises.md`: 综合实战与性能优化
