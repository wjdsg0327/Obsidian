# 第 0 章练习：环境准备与第一段 OpenCV 代码

## 学习目标

- 确认 `opencv-python` 和 `numpy` 已正确安装。
- 理解图像是 `numpy.ndarray` 这一基本概念。
- 学会用 `cv2.imwrite()` 保存第一张测试图像。

## 练习题

### 基础练习 1

编写一个脚本，打印 OpenCV 版本号、NumPy 版本号，并生成一张 300x300 的纯黑图。

### 基础练习 2

在纯黑图中心写上你的名字拼音，并保存成 `my_first_opencv.png`。

### 进阶练习

把背景色改成深蓝色，文字颜色改成亮黄色，再额外画一条对角线。

### 思考题

为什么 OpenCV 图像对象适合用 NumPy 来表示？请从数据结构角度简单总结。
